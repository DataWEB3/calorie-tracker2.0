# CalorieTrack Pro 🥗

Современное приложение для трекинга калорий и тренировок с премиальным дизайном.

## ✨ Функции

- **Дашборд** — кольцевой прогресс КБЖУ в реальном времени
- **Дневник питания** — база 40+ продуктов, поиск, добавление своих
- **Планировщик тренировок** — алгоритм подбора 3-5 упражнений
- **Расчёт нормы** — формула Миффлина-Сан Жеора

## 🛠 Технологии

- React 18 + Vite
- Tailwind CSS (премиальный дизайн)
- Zustand (управление состоянием)
- Framer Motion (анимации)
- Lucide React (иконки)

## 🚀 Запуск

```bash
cd calorie-tracker
npm install
npm run dev
```

## 📐 Формула Миффлина-Сан Жеора

**BMR (мужчины):** 10 × вес(кг) + 6.25 × рост(см) − 5 × возраст + 5  
**BMR (женщины):** 10 × вес(кг) + 6.25 × рост(см) − 5 × возраст − 161

**TDEE = BMR × коэффициент активности**
